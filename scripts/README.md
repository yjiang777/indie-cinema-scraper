# Utility Scripts

One-time maintenance and debugging scripts.

## Maintenance Scripts

- **enrich_movies_with_tmdb.py** - Add TMDB metadata (directors, runtime, etc)
- **fix_special_cases.py** - Handle double features and TV shows
- **check_missing_directors.py** - List movies without directors

## Debug Scripts

- **debug_*.py** - Theater structure investigation scripts

## Usage

Run from project root:
```bash
python scripts/enrich_movies_with_tmdb.py
```
